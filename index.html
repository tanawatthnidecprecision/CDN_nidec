เริ่มต้น

CDN (Content Delivery Network) เป็นระบบที่ออกแบบมาเพื่อเพิ่มประสิทธิภาพและลดความล่าช้าในการโหลดเนื้อหาบนเว็บไซต์ โดยการกระจายเนื้อหา (content) ไปยังเซิร์ฟเวอร์ที่ตั้งอยู่ใกล้กับผู้ใช้สุด

ทำให้เว็บไซต์ทำงานได้เร็วขึ้นและมีประสิทธิภาพดีกว่า. หลักการทำงานของ CDN คือการแจกจ่ายเนื้อหาไปยังเซิร์ฟเวอร์ที่ตั้งอยู่ในตำแหน่งทางภูมิศาสตร์ที่เหมาะสมกับผู้ใช้. เมื่อมีคำขอ (request) จากผู้ใช้,

CDN จะนำเนื้อหาจากเซิร์ฟเวอร์ที่อยู่ใกล้ที่สุดมาให้ ซึ่งลดการล่าช้าในการโหลดข้อมูลเนื้อหา.

ปุ่มสำหรับใช้งาน

ปุ่ม (button) เป็นองค์ประกอบที่สามารถใช้ในการสร้างตัวเลือกที่ผู้ใช้สามารถคลิกหรือแตะเพื่อทำการกระทำหรือเปิดใช้งานฟังก์ชันต่าง ๆ บนเว็บไซต์หรือแอปพลิเคชันเว็บ.






Input และ Select

ใน HTML, input มีหลายประเภทที่สามารถใช้ในการรับข้อมูลจากผู้ใช้:

input type="text"

ใช้สำหรับรับข้อมูลจากผู้ใช้ในรูปแบบข้อความ.

input type="password":

ใช้เหมือนกับ input type="text" แต่จะซ่อนตัวอักษรที่ผู้ใช้ป้อน, เหมาะสำหรับการป้องกันการดูรหัสผ่าน.

select:

ใช้สร้างกล่องคำสั่ง (dropdown list) ที่ผู้ใช้สามารถเลือกรายการได้. มักใช้ร่วมกับ option เพื่อระบุรายการที่สามารถเลือกได้.

Switch

สวิตช์นี้มักถูกออกแบบเพื่อให้ผู้ใช้สามารถเปิดหรือปิดอุปกรณ์ที่เชื่อมต่อกับ IoT ได้ง่ายและสะดวก.

ตัวอย่าง UI สวิตช์ที่เกี่ยวกับ IoT อาจมีลักษณะดังนี้:

ปุ่มสวิตช์ (Switch Button)

ปุ่มที่ผู้ใช้สามารถคลิกหรือแตะเพื่อเปิดหรือปิดอุปกรณ์. ตัวอย่างเช่นปุ่ม ON/OFF.

สถานะอุปกรณ์ (Device Status)

แสดงสถานะปัจจุบันของอุปกรณ์ว่าเปิดหรือปิดอยู่.

การแสดงผลข้อมูล (Data Display)

แสดงข้อมูลที่เกี่ยวข้องกับอุปกรณ์, เช่นอุณหภูมิปัจจุบัน, ความชื้น, หรือข้อมูลเซ็นเซอร์.

เมนูบาร์

เป็นแท็กใน HTML ที่ใช้เพื่อระบุส่วนของเว็บไซต์ที่เป็นเมนูนำทางหรือเมนูการนำทาง. นั่นคือ,

nav จะใช้ร่วมกับลิงก์ที่ชี้ไปยังหน้าเว็บต่าง ๆ หรือส่วนต่าง ๆ ของเว็บไซต์ที่ให้ผู้ใช้ไปที่ส่วนต่าง ๆ ของเนื้อหาหรือฟีเจอร์พิเศษ.

Grid

CSS Grid (Grid Layout) เป็นระบบที่ถูกออกแบบมาเพื่อจัดการและจัดวางองค์ประกอบต่าง ๆ ในหน้าเว็บอย่างยืดหยุ่นและมีประสิทธิภาพ. ด้วย CSS Grid, คุณสามารถสร้างโครงสร้างเว็บที่ซับซ้อนได้โดยไม่จำเป็น

ต้องพึ่งพาการจัดวางด้วย Flexbox หรือ float ซึ่งอาจทำให้การจัดการหน้าเว็บเป็นไปได้ยากและซับซ้อน.
